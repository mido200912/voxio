# ⚙️ التفاصيل التقنية للباك-إيند (Backend Deep Dive)

يعمل الباك-إيند كمحرك مركزي لخدمات VOXIO، وهو مبني باستخدام **Node.js** و **Express**.

## 🛠️ التكنولوجيات المستخدمة (Tech Stack)
- **Node.js & Express**: لإدارة السيرفر والمسارات.
- **Firebase Admin SDK**: للتفاعل مع قاعدة بيانات Firestore و Authentication.
- **JWT (JSON Web Tokens)**: لتأمين جلسات المستخدمين.
- **Helmet & HPP**: لحماية السيرفر من الهجمات الشائعة.
- **CORS**: لإدارة السماح بالوصول من الدومينات المختلفة.

## 🔑 الميزات الرئيسية (Core Features)

### 1. نظام التدريب والذكاء الاصطناعي (`/api/ai`)
- يتعامل مع رفع الملفات واستخراج النصوص.
- يربط بيانات الشركة بالذكاء الاصطناعي لإنشاء "شخصية" مخصصة للرد.

### 2. نظام التكاملات (`/api/integrations`)
- **Telegram Webhook**: استقبال وإرسال الرسائل عبر تليجرام.
- **WhatsApp**: التكامل مع API واتساب (أو خدمات وسيطة).
- **Instagram**: الرد الآلي على الرسائل والتعليقات.

### 3. ويدجت الدردشة (`/widget.js`)
- السيرفر يقدم ملف Javascript يمكن تضمينه في أي موقع.
- هذا الملف يقوم بإنشاء Iframe يتصل بـ VOXIO للسماح للزوار بالدردشة مع البوت.

### 4. إدارة الشركات والبيانات (`/api/company`)
- تخزين رؤية الشركة، مهمتها، وقيمها.
- استخدام هذه البيانات لتوجيه ردود البوت (Custom Instructions).

## 🛡️ الأمان (Security)
- يتم استخدام `rateLimit` لمنع الهجمات الغاشمة (Brute Force) على مسارات الدخول.
- يتم تطهير البيانات المدخلة (XSS Sanitization) لمنع حقن الأكواد الخبيثة.
- جميع المسارات الحساسة محمية بـ Middleware يتحقق من الـ Token.
