# 🎨 التفاصيل التقنية للفرونت-إيند (Frontend Deep Dive)

واجهة VOXIO مبنية لتكون سريعة، تفاعلية، وسهلة الاستخدام، باستخدام **React** و **Vite**.

## 🛠️ التكنولوجيات المستخدمة (Tech Stack)
- **React.js**: مكتبة بناء الواجهات.
- **React Router Dom**: لإدارة التنقل بين الصفحات.
- **Context API**: لإدارة الحالة (Global State) مثل:
  - `AuthContext`: بيانات المستخدم المسجل.
  - `ThemeContext`: التبديل بين الوضع المظلم والمضيء.
  - `LanguageContext`: دعم تعدد اللغات (العربية والإنجليزية).
- **CSS Modules / Vanilla CSS**: للتنسيق الجمالي مع دعم الـ Responsive.

## 📱 أقسام الواجهة الرئيسية

### 1. لوحة تحكم المستخدم (User Dashboard)
- **Home**: نظرة عامة على الإحصائيات (عدد الرسائل، الحالات النشطة).
- **AI Training**: واجهة لرفع الملفات، إضافة روابط المواقع، وكتابة النصوص لتدريب البوت.
- **Integrations**: صفحة لربط المنصات المختلفة (تليجرام، واتساب، فيسبوك).
- **Settings**: تعديل بيانات الشركة، الشعار، وإعدادات الملف الشخصي.

### 2. محرر البوت (Chatbot Editor)
- واجهة مخصصة لتجربة ردود البوت وتعديل التعليمات البرمجية (System Prompts) بشكل لحظي.

### 3. ويدجت الدردشة (Chat Widget)
- صفحة منفصلة (`/widget/:apiKey`) مصممة لتظهر داخل Iframe في مواقع العملاء.
- تتميز بتصميم عصري (Premium Design) يدعم الرموز التعبيرية والردود السريعة.

## 🌐 دعم اللغات (Localization)
- المشروع يدعم بالكامل اللغتين **العربية والإنجليزية**.
- يتم استخدام ملف `translations.js` لتخزين النصوص المترجمة.
- الواجهة تنقلب بالكامل (RTL/LTR) بناءً على اللغة المختارة.
